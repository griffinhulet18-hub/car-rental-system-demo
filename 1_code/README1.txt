Car Rental System
Author: Griffin Hulet

How to Run the Project

1. Install Java JDK 17 or later.
2. Install Maven if it is not already installed.
3. Open the project in IntelliJ IDEA or Eclipse.
4. Navigate to the project folder:
   Car_Rental_System_With_POM
5. Run the Application.java file.

OR run from terminal:

mvn spring-boot:run

6. Open a browser and go to:

http://localhost:8081

The system will load the Car Rental System homepage.

Technologies Used
- Java Spring Boot
- Maven
- H2 Database
- Thymeleaf
